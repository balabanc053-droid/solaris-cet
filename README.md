# 🌞 Solaris-CET

**Solaris-CET** este un asistent AI pentru energie verde, panouri solare și tehnologii sustenabile.

## ✨ Caracteristici

- 🤖 **AI Integrat** - Rulează direct în browser folosind Transformers.js și TinyLlama
- 💯 **100% Gratuit** - Fără chei API, fără costuri lunare
- 🔒 **Privat** - Datele tale nu pleacă niciodată de pe dispozitiv
- ⚡ **Rapid** - Model optimizat pentru browser (60MB)
- 🌐 **Disponibil Online** - Găzduit gratuit pe GitHub Pages

## 🚀 Tehnologii Utilizate

- React 18 + TypeScript
- Transformers.js (Hugging Face)
- TinyLlama 1.1B Chat Model
- GitHub Pages (Hosting)

## 🛠️ Dezvoltare Locală

```bash
# Clonează repository-ul
git clone https://github.com/aamclaudiu-hash/solaris-cet.git

# Intră în director
cd solaris-cet

# Instalează dependențele
npm install

# Pornește serverul de dezvoltare
npm start
